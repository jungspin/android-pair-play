# Pair Play
이름과 맥주소로 블루투스 디바이스를 손쉽게 페어링 할 수 있어요!

## 사용언어
* kotlin

## 개발 배경
- 많은 블루투스 디바이스를 페어링 해야했는데, 디바이스의 이름이 모두 동일해 어려움을 겪었습니다.
- 고유한 맥주소를 알 수 있으면 보다 편리할 것 같아 만들게 되었습니다.
